Project Scope: The purpose of this course project is to create a ride analytics database to track trips, ride fares, and demand surges, whcich are sudden spikes in ride-sharing activity. In addition, the database must include measures to detect these demand surges, along with time-based performance optimization.

The stakeholders involved include ride-sharing companies such as Uber and Lyft, as well as investors that have a share of these companies. Customers are also classified as steakholders as well because they are the main users of ride-sharing applications.
